# README

## Dependencies

1. ruby 3.0.2
2. Rails 6.1.4
3. Postgresql 12.3


## Setup locally

1. Clone the repo

`git clone git@github.com:simaob/scoreboard.git`

2. Run bundle install

`bundle install`

3. Run yarn install

`yarn install`

4. Create database

`bundle exec rails db:create`

5. Run migrations

`bundle exec rails db:migrate`

6. Run the server

`bundle exec rails server`
